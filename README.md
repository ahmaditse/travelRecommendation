# Travel Recommendation Website

This project is a travel recommendation web application created as part of the IBM JavaScript course.

## Features
- Navigation bar with Home, About, Contact
- Search recommendations (Beach, Temple, Country)
- Dynamic data using Fetch API
- Country local time display
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON
- GitHub Pages
